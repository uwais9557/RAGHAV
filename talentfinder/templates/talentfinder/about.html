About Talent Finder

An AI-powered Smart Resume Analyzer and Job Recommendation System built with Django and modern web technologies.

Project Overview

Talent Finder is a comprehensive web application that leverages Natural Language Processing, TF-IDF vectorization, and cosine similarity algorithms to intelligently analyze resumes and match candidates with the most relevant job opportunities.

The system automatically extracts skills from PDF, DOCX, TXT, and image files using OCR technology, calculates an ATS compatibility score, and provides personalized job recommendations from a database of top Indian tech companies.
